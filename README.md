# FileXORProcessor
Qt/C++ приложение для Windows: многопоточный инструмент для пакетной XOR-модификации файлов с поддержкой обработки больших объёмов данных (10GB+).

### Требования
- **ОС:** Windows 7/10/11
- **Qt:** 6.x или 5.15+
- **Компилятор:** MinGW-w64 или MSVC
- **CMake:** 3.16+
